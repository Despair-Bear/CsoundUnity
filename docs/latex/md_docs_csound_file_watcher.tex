From version 3.\+0 \mbox{\hyperlink{class_csound_unity}{Csound\+Unity}} can detect the changes made to the {\itshape csd} file while modifying it in an external editor. For this to work add {\bfseries{FILEWATCHER\+\_\+\+ON}} in your project Scripting Define Symbols. {\bfseries{This is an experimental feature}}\+: please report any issue \href{https://github.com/rorywalsh/CsoundUnity/issues}{\texttt{ on Github}}.
