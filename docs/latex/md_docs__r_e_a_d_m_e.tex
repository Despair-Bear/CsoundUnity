\mbox{\hyperlink{class_csound_unity}{Csound\+Unity}} is a custom \mbox{\hyperlink{namespace_csound}{Csound}} package for Unity. It provides Unity users with a means of accessing \mbox{\hyperlink{namespace_csound}{Csound}}\textquotesingle{}s core API within their Unity C\# scripts. Borrowing from Richard Henninger\textquotesingle{}s Csound6 .Net framework, \mbox{\hyperlink{class_csound_unity}{Csound\+Unity}} provides a Unity bridge to a set of p\+Invoke signatures that are used to call \mbox{\hyperlink{namespace_csound}{Csound}}\textquotesingle{}s native API functions from C\# code. The system works on OSX, Windows and Android.
